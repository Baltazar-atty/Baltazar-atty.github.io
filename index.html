¿Qué es Programación?

La programación es el proceso de crear instrucciones que una computadora puede entender para ejecutar tareas específicas. Se utilizan distintos lenguajes como Python, Java, C++, entre otros.

Lenguajes más comunes

  • Python
  • Java
  • C#
  • JavaScript
  • HTML y CSS
